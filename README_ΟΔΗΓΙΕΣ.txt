AI Underwriter PWA v2.0 - Οδηγίες

ΣΕ ΥΠΟΛΟΓΙΣΤΗ
1. Κάνε unzip τον φάκελο.
2. Άνοιξε το index.html με διπλό κλικ, ή για καλύτερη λειτουργία τρέξε το START_AI_UNDERWRITER_MAC.command.

ΣΕ iPHONE ΣΤΟ ΙΔΙΟ Wi-Fi
1. Στον Mac άνοιξε το START_AI_UNDERWRITER_MAC.command.
2. Βρες το IP του Mac: System Settings > Wi‑Fi > Details > IP Address.
3. Στο iPhone άνοιξε Safari και γράψε: http://IP-ΤΟΥ-MAC:8000
   Παράδειγμα: http://192.168.1.15:8000
4. Πάτα Share > Add to Home Screen για να εμφανίζεται σαν εφαρμογή.

Σημείωση: Για πλήρες offline PWA στο iPhone, ιδανικά χρειάζεται HTTPS φιλοξενία. Τοπικά μέσω Wi‑Fi λειτουργεί για χρήση στον browser.
